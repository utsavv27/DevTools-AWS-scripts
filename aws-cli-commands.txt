### General AWS CLI Commands
- **Configure AWS CLI**:  

  aws configure
 
  Configures AWS CLI with access keys, region, and output format.

- **View Help**:  

  aws help
  aws <service> help

  Displays help for AWS CLI commands or specific services.

### Amazon S3 (Simple Storage Service)
- **List Buckets**:  

  aws s3 ls
  ```

- **Create a Bucket**:  
  ``` h
  aws s3 mb s3://my-bucket-name
  ```

- **Upload a File**:  
  ``` a
  aws s3 cp myfile.txt s3://my-bucket-name/
  ```

- **Download a File**:  
  ``` a
  aws s3 cp s3://my-bucket-name/myfile.txt ./
  ```

- **Sync Directories**:  
  ``` a
  aws s3 sync /local/dir s3://my-bucket-name/dir
  ```

### Amazon EC2 (Elastic Compute Cloud)
- **List Instances**:  
  ``` a
  aws ec2 describe-instances
  ```

- **Start an Instance**:  
  ``` a
  aws ec2 start-instances --instance-ids i-1234567890abcdef0
  ```

- **Stop an Instance**:  
  ``` a
  aws ec2 stop-instances --instance-ids i-1234567890abcdef0
  ```

- **Reboot an Instance**:  
  ``` a
  aws ec2 reboot-instances --instance-ids i-1234567890abcdef0
  ```

- **Terminate an Instance**:  
  ``` a
  aws ec2 terminate-instances --instance-ids i-1234567890abcdef0
  ```

### Amazon RDS (Relational Database Service)
- **List Databases**:  
  ``` a
  aws rds describe-db-instances
  ```

- **Create a Database**:  
  ``` a
  aws rds create-db-instance --db-instance-identifier mydbinstance --allocated-storage 20 --db-instance-class db.t2.micro --engine mysql --master-username admin --master-user-password password
  ```

- **Delete a Database**:  
  ``` a
  aws rds delete-db-instance --db-instance-identifier mydbinstance --skip-final-snapshot
  ```

- **Describe EKS Cluster**:  
  ``` a

  ```

### AWS CloudWatch
- **List Metrics**:  
  ``` a
  aws cloudwatch list-metrics
  ```

- **Get Metric Statistics**:  
  ``` a
  aws cloudwatch get-metric-statistics --namespace AWS/EC2 --metric-name CPUUtilization --dimensions Name=InstanceId,Value=i-1234567890abcdef0 --start-time 2024-08-21T23:18:00Z --end-time 2024-08-22T23:18:00Z --period 3600 --statistics Average
  ```

  ``` a
